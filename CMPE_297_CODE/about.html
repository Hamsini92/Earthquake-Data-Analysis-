This is a simple implementation of Shiny dashboard to explore the Big Data Conecpt using United State Earthquakes data. The earthquake data used here are near real-time. The records were downloaded from USGS website (csv format) and placed in the data folder. To filter the quakes in the vicinity of United States, only the quakes within the bounding box of United States are used.

You need following packages (if you have RStudio installed)to run this application locally.

R packages used

  • Shiny
  • shinydashboard
  • leaflet
  • dplyr
  • scales
  • ggplot2
  • reshape2
  • data.table
  • rgdal
  • jsonlite
